## Welcome to GeoChallenge

GeoChallenge is a real-time geography trivia game that tests your knowledge of world geography through interactive challenges. Whether you're identifying flags, locating countries on a map, naming capitals, or recognizing country silhouettes, GeoChallenge offers an engaging way to learn and compete.

## Key Features

### Multiple Question Categories

GeoChallenge offers five distinct question categories to test different aspects of your geography knowledge:

<AccordionGroup>
  <Accordion title="Flags" icon="flag">
    Identify countries by their national flags. Each flag question presents you with a flag image and four country options. Perfect for visual learners and vexillology enthusiasts.
  </Accordion>
  
  <Accordion title="Capitals" icon="landmark">
    Test your knowledge of world capitals. Given a country, identify its capital city from multiple choices. Essential knowledge for any geography buff.
  </Accordion>
  
  <Accordion title="Maps" icon="map">
    Click on an interactive map to locate specific countries. Uses Leaflet for smooth pan and zoom controls, with distance-based scoring for accuracy.
  </Accordion>
  
  <Accordion title="Silhouettes" icon="image">
    Recognize countries by their distinctive shapes and outlines. A unique challenge that tests your spatial memory and geographical awareness.
  </Accordion>
  
  <Accordion title="Mixed" icon="dice">
    A randomized combination of all categories. The ultimate test that keeps you on your toes with varied question types.
  </Accordion>
</AccordionGroup>

## How Scoring Works

<Note>
  GeoChallenge uses a dynamic scoring system that rewards both accuracy and speed.
</Note>

Each correct answer is worth up to **150 points**:
- **Base points**: 100 points for a correct answer
- **Time bonus**: Up to 50 additional points based on how quickly you answer
- **Map questions**: Distance-based scoring with closer guesses earning more points

```typescript
// Scoring calculation from backend
const basePoints = 100;
const maxTimeBonus = 50;
const timeBonus = Math.floor((timeRemaining / totalTime) * maxTimeBonus);
const totalPoints = basePoints + timeBonus;
```

## Mobile-First Design

GeoChallenge is built with a mobile-first philosophy, ensuring an optimal experience on all devices:

- **Responsive layout** using Tailwind CSS with `dvh` units for proper mobile viewport handling
- **Touch-optimized** interactions with swipe gestures for category selection
- **Safe area insets** for devices with notches and dynamic islands
- **Offline-ready** with PWA support and service workers
- **Dark mode** with automatic theme detection via `prefers-color-scheme`

## Real-Time Multiplayer

Powered by Socket.IO, GeoChallenge delivers seamless real-time multiplayer experiences:

- **Instant matchmaking** for duels based on selected category
- **Live updates** during head-to-head matches
- **Synchronized questions** ensuring fair competition
- **Graceful disconnection handling** with timeout mechanisms
- **Redis-backed** session management for scalability
