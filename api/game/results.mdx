---
title: Finish Game
api: POST /api/game/finish
description: Complete a game session and save results
---

## Authentication

Required. Must provide a valid JWT token:

```
Authorization: Bearer <token>
```

## Request Body

<ParamField body="answers" type="array" required>
  Array of all answers submitted during the game
  
  <ParamField body="questionId" type="string" required>
    Question identifier (CUID)
  </ParamField>
  
  <ParamField body="answer" type="string" required>
    User's submitted answer
  </ParamField>
  
  <ParamField body="timeRemaining" type="number" required>
    Time remaining when answer was submitted (in seconds)
    
    Must be between 0 and the time limit per question (typically 10 seconds)
  </ParamField>
  
  <ParamField body="coordinates" type="object" optional>
    Click coordinates for MAP questions only
    
    <ParamField body="lat" type="number">
      Latitude of clicked location
    </ParamField>
    
    <ParamField body="lng" type="number">
      Longitude of clicked location
    </ParamField>
  </ParamField>
</ParamField>

<ParamField body="category" type="string" optional>
  Game category for statistics tracking
  
  Possible values: `FLAG`, `CAPITAL`, `MAP`, `SILHOUETTE`, `MIXED`
</ParamField>

## Response

<ResponseField name="message" type="string">
  Success message ("Partida finalizada")
</ResponseField>

<ResponseField name="gameId" type="string">
  Unique identifier for the saved game result (CUID)
</ResponseField>

<ResponseField name="totalScore" type="number">
  Sum of all points earned across all answers
  
  Maximum possible: 1,500 points (10 questions × 150 points each)
</ResponseField>

<ResponseField name="correctCount" type="number">
  Number of correct answers in the game
</ResponseField>

<ResponseField name="totalQuestions" type="number">
  Total number of questions answered
</ResponseField>

<ResponseField name="accuracy" type="number">
  Percentage of correct answers (rounded to integer)
  
  Formula: `Math.round((correctCount / totalQuestions) * 100)`
</ResponseField>

<ResponseField name="isHighScore" type="boolean">
  Whether this score is the user's new personal best
  
  - If `true`, the user's `highScore` field is updated
  - If `true`, the leaderboard is also updated
</ResponseField>

<ResponseField name="details" type="array">
  Detailed breakdown of each answer
  
  <ResponseField name="questionId" type="string">
    The question identifier
  </ResponseField>
  
  <ResponseField name="isCorrect" type="boolean">
    Whether this answer was correct
  </ResponseField>
  
  <ResponseField name="correctAnswer" type="string">
    The correct answer for this question
  </ResponseField>
  
  <ResponseField name="userAnswer" type="string">
    The user's submitted answer
  </ResponseField>
  
  <ResponseField name="points" type="number">
    Points earned for this answer (0-150)
  </ResponseField>
  
  <ResponseField name="distance" type="number" optional>
    Distance in kilometers from target (MAP questions only)
    
    Used to determine correctness and calculate score
  </ResponseField>
  
  <ResponseField name="timeRemaining" type="number">
    Time remaining when answer was submitted (seconds)
  </ResponseField>
</ResponseField>

## Error Responses

<ResponseField name="400 Bad Request">
  - Invalid request body structure
  - Missing required fields
  - Invalid time values (negative or exceeding limit)
</ResponseField>

<ResponseField name="401 Unauthorized">
  Missing or invalid JWT token
</ResponseField>

<ResponseField name="500 Internal Server Error">
  Database or server error during game save
</ResponseField>

## Scoring Mechanics

Each answer is scored individually and then summed:

### Multiple-Choice Questions (FLAG, CAPITAL, SILHOUETTE)

```typescript
// Correct answer
const basePoints = 100;
const maxTimeBonus = 50;
const timeBonus = Math.floor((timeRemaining / totalTime) * maxTimeBonus);
const points = basePoints + timeBonus; // 100-150 points
```

### MAP Questions

Distance-based scoring with correctness threshold at 500km:

```typescript
const distance = haversineDistance(userCoords, targetCoords);
const isCorrect = distance <= 500;

if (distance <= 500) {
  // Very accurate
  points = basePoints + timeBonus;
} else {
  // Apply accuracy penalty
  const accuracyFactor = Math.max(0, 1 - distance / 2000);
  points = Math.floor((basePoints + timeBonus) * accuracyFactor);
}
```

## Side Effects

When a game is completed successfully:

1. **Game result saved** to `GameResult` table in database
2. **User stats updated**:
   - `gamesPlayed` incremented by 1
   - `highScore` updated if new score is higher
3. **Leaderboard updated** if new high score (via `updateLeaderboardScore`)

From `backend/src/controllers/game.controller.ts:135-192` and `backend/src/services/game.service.ts`.
