---
title: Get User Leaderboard Context
api: GET /api/leaderboard/me
description: Retrieve the authenticated user's rank with surrounding players for context
---

## Authentication

Required. Must provide a valid JWT token.

## Query Parameters

No query parameters. The context window (3 players above and below) is configured server-side.

## Response

<ResponseField name="userRank" type="object | null">
  Current user's rank information. `null` if the user has no score yet.
  
  <Expandable title="LeaderboardEntry properties">
    <ResponseField name="rank" type="number">
      User's position in the leaderboard (1-indexed)
    </ResponseField>
    
    <ResponseField name="userId" type="string">
      User's unique identifier
    </ResponseField>
    
    <ResponseField name="username" type="string">
      User's display name
    </ResponseField>
    
    <ResponseField name="score" type="number">
      User's highest score
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="neighbors" type="array">
  Array of nearby players (up to 3 above and 3 below the user)
  
  <Expandable title="LeaderboardEntry properties">
    <ResponseField name="rank" type="number">
      Player's position in the leaderboard
    </ResponseField>
    
    <ResponseField name="userId" type="string">
      Player's unique identifier
    </ResponseField>
    
    <ResponseField name="username" type="string">
      Player's display name
    </ResponseField>
    
    <ResponseField name="score" type="number">
      Player's highest score
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Informational message (only present when user has no score)
</ResponseField>

## Example Request

```bash
curl -X GET "https://api.geochallenge.com/api/leaderboard/me" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

## Error Responses

<ResponseField name="error" type="string">
  Error message when the request fails
</ResponseField>

### 401 Unauthorized

Returned when no valid authentication token is provided.

### 500 Internal Server Error

```json
{
  "error": "Error interno del servidor"
}
```
